# SchemasAura — System Requirements

Техническое задание на разработку веб-приложения для совместного моделирования бизнес-процессов в реальном времени.

## О проекте

SchemasAura — платформа для командной работы с диаграммами бизнес-процессов (BPMN, IDEF0, DMN) с поддержкой совместного редактирования в реальном времени. ТЗ разработано в соответствии с ГОСТ 34.602-2020.

## Что сделано

- Разработал полное **техническое задание по ГОСТ 34.602-2020**
- Описал роли пользователей и их сценарии работы
- Сформулировал **функциональные требования** к системе
- Описал **нефункциональные требования** (производительность, безопасность, надёжность)
- Составил требования к интерфейсу и совместимости
- Описал требования к безопасности и защите данных

## Основные функции системы

- Создание и редактирование диаграмм в нотациях BPMN, IDEF0, DMN
- Совместное редактирование в реальном времени
- Управление версиями диаграмм
- Экспорт в форматы PDF, PNG, XML
- Ролевая модель доступа (администратор, редактор, читатель)

## Стандарты

`ГОСТ 34.602-2020` `BPMN 2.0` `IDEF0` `DMN`
